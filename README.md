# Backdrop

An Android gallery app for showcasing the AndroidLiquidGlass / Backdrop Compose UI effects.

The app includes movable background images, adjustable glass parameters for each showcase control, and copyable Kotlin snippets for the current style.

## CI build

Pushing to `main` runs `.github/workflows/android.yml` and uploads a fixed-signature release APK artifact.

The committed keystore is a demo signing key used only to keep generated APKs install-compatible with earlier CI builds.
